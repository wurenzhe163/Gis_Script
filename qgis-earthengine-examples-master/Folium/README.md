# Earth Engine Python API and Folium Interactive Mapping

Jupyter notebook examples for using the [Google EarthEngine Python API](https://developers.google.com/earth-engine/python_install) and [folium](https://github.com/python-visualization/folium) library for interactive mapping.

* [Run in binder](https://mybinder.org/v2/gh/giswqs/qgis-earthengine-examples/master?filepath=Folium%2Fee-api-folium-setup.ipynb)
